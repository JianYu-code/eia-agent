# K14 公众参与审核要点（报告书适用）

## 依据
《环境影响评价公众参与办法》（生态环境部令第4号，2019年1月1日施行）

## 程序要求
1. **首次信息公开**：确定报告书编制单位后 7 个工作日内，网络平台公示（项目名称、选址、建设内容、建设单位联系方式）
2. **征求意见稿公示**：报告书征求意见稿完成后，
   - 网络平台公示 ≥10 个工作日
   - 报纸公示 ≥2 次
   - 张贴公告（项目所在地公众易于知悉的场所）≥10 个工作日
3. **报批前公示**：报批前网络平台公开拟报批报告书全文和公众参与说明
4. **公众意见处理**：对意见采纳与否逐条说明，不采纳须说明理由
5. **深度公参**：对环境影响方面公众质疑性意见多的项目，应组织听证会/座谈会

## 审核要点
- 报告书无公众参与章节或无公参说明 → P0
- 公示次数/期限不足 → P1
- 无公示证明材料（截图、报纸原件）→ P1
- 公众意见未逐条回应 → P1
- 报告表、登记表项目不强制公参，做了不扣分
