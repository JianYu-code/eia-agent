# K11 预测模型与预测内容审核要点

## 大气预测模型适用条件（HJ 2.2-2018）
- AERMOD：点源/面源/线源，评价范围 ≤50km，简单地形
- ADMS：同 AERMOD，适合复杂下垫面（城市）
- CALPUFF：评价范围 >50km、复杂风场（ coastal / 山谷）、二次污染物
- CMAQ/CAMx：区域尺度光化学网格模型
- 岸边熏烟、建筑物下洗等特殊情形须勾选对应选项

## 预测内容（按等级）
- 一级：全年逐时/逐日气象，正常+非正常工况，短期+长期浓度，叠加背景值，大气环境防护距离计算
- 二级：可简化气象，正常工况短期/长期浓度，防护距离
- 三级：不进行进一步预测，只给估算结果

## 其他要素
- 地表水：河流一维/二维模型、湖库模型选择与水文条件匹配
- 地下水：解析法/数值法，须进行包气带防护性能分析
- 噪声：点声源/线声源衰减模式，室内声源等效室外化
- 生态：三级可定性分析

## 审核要点
1. 模型选择与项目条件不匹配 → P1
2. 气象数据年代（近 20 年统计）与站点代表性 → P1
3. 预测因子与评价因子不一致 → P1
4. 未叠加现状背景浓度 → P1
5. 大气防护距离未计算或计算后仍有敏感点未说明 → P0
6. 预测结果与结论矛盾（预测超标但结论可行）→ P0
