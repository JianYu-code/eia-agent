# K07 报告结构完整性审核要点

## 报告书必备章节（HJ 2.1-2016）
1. 前言（任务由来、评价工作过程、主要结论）
2. 总则（编制依据、评价因子、评价标准、评价等级、评价范围、环境保护目标）
3. 建设项目工程分析（现有工程回顾+拟建工程）
4. 环境现状调查与评价
5. 环境影响预测与评价
6. 环境保护措施及其可行性论证
7. 环境影响经济损益分析
8. 环境管理与监测计划
9. 环境影响评价结论

## 报告表必备内容
1. 建设项目基本情况
2. 建设项目工程分析
3. 区域环境质量现状、环境保护目标及评价标准
4. 主要环境影响和保护措施
5. 环境保护措施监督检查清单
6. 结论
7. 附表（污染物排放量汇总表）

## 审核要点
1. 缺必备章节 → P0（注意章节标题允许同义表述，如"环境保护措施"也可写作"污染防治措施"）
2. 章节有标题但内容空洞（<200 字或仅"见附图"）→ P1
3. 报告表混入报告书章节体系（或反之）→ 报告类型识别问题 → P1
4. 附件清单：附图、附表、附件（立项文件、委托书、监测报告、公众参与材料）→ 缺失 P1
5. 编制单位资质页、编制主持人信息（信用平台登记）→ 缺失 P1
6. 页码、目录完整性 → P2

## 内容自洽性
- 结论章节与正文预测结果必须一致
- 各章引用的标准版本必须统一
- 项目名称、建设单位、建设地点全文一致
