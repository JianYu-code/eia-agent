# K13 总量控制审核要点

## 依据
《建设项目主要污染物排放总量指标审核及管理暂行办法》（环发〔2014〕197号）

## 总量控制因子
- 废水：COD、氨氮、总氮、总磷（部分流域）
- 废气：SO₂、NOx、烟粉尘、VOCs
- 重金属：铅、汞、镉、铬、砷（重点行业）

## 审核要点
1. **指标来源**：是否取得生态环境主管部门下达的总量指标（或排污权交易凭证）→ 缺失 P1
2. **倍量削减**：环境质量超标区域，新增污染物须倍量替代（2倍）；达标区域等量替代
3. **三本账一致性**：改扩建项目"以新带老"削减量与现有工程排放量逻辑一致
4. **计算口径**：总量按年排放量（t/a）核算，与预测用小时最大排放量口径不同
5. **VOCs 总量**：有组织+无组织合计
6. 报告表中污染物排放量汇总表与正文数据一致

## 常见错误
- 总量指标数值与批复文件不符 → P0
- 未说明替代来源（区域削减方案）→ P1
- 只算有组织漏算无组织 VOCs → P1
