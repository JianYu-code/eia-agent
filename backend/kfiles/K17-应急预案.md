# K17 应急预案审核要点

## 依据
《企业事业单位突发环境事件应急预案备案管理办法（试行）》（环发〔2015〕4号）
《企业突发环境事件风险分级方法》（HJ 941-2018）
《企业突发环境事件风险评估指南（试行）》

## 备案资料包完整性（七件套）
1. 突发环境事件应急预案备案表
2. 环境应急预案及编制说明
3. 环境风险评估报告
4. 环境应急资源调查报告
5. 环境应急预案评审意见
6. 发布令（签署发布文件）
7. 修改说明（评审后修改情况）

## 预案内容审核
1. **风险评估**：风险物质识别（Q 值计算）、风险源清单、最大可信事故、风险等级（一般/较大/重大）→ 缺 Q 值表 P0
2. **组织体系**：应急指挥部、各应急组职责、24 小时应急电话
3. **分级响应**：车间级/企业级/社会级响应条件与启动程序
4. **处置措施**：污染源切断、围堵收集、应急监测（点位/因子/频次）、人员疏散路线
5. **应急保障**：物资清单（名称/数量/位置）、经费、通讯、医疗
6. **预案管理**：演练频次（每年至少 1 次）、评估修订（每 3 年或重大变化时）
7. **附件**：应急物资分布图、疏散路线图、应急通讯录、危废处置协议

## 与环评联动
- 风险物质清单应与环评报告一致；新改扩建后预案应同步修订
- 涉危废企业应将危废流失/泄漏纳入情景

## 常见缺陷
- 预案套用模板未结合企业实际（出现其他企业名称）→ P0
- 应急物资清单无数量/位置 → P1
- 无应急监测方案 → P1
- 演练记录缺失 → P2
