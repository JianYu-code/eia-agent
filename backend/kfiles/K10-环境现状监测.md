# K10 环境现状监测审核要点

## 监测方案
1. **布点**：点位数量/位置须满足导则要求（大气一级≥7个、二级≥6个；厂界噪声四周+敏感点）
2. **因子**：常规因子+特征因子全覆盖；特征因子须与评价因子对应
3. **频次**：大气连续 7 天（小时值每天 4 次）；地表水丰/枯水期；噪声昼夜各一次
4. **时效**：监测数据 3 年内有效；引用数据须说明来源与代表性
5. **资质**：监测报告须有 CMA 章；监测单位与报告编号可查

## 数据有效性
1. 检出限与单位标注；未检出的表示方法
2. 气象条件记录（风向、风速、温度）
3. 工况记录（监测期间生产负荷）
4. 质控措施（平行样、加标回收）

## 现状评价
1. 与相应质量标准对标（功能区划要正确！）
2. 超标因子分析原因
3. 区域环境质量变化趋势（报告书要求）
4. 引用数据与项目位置的距离合理性（大气引用≤50km 内同类区域）

## 常见缺陷
- 现状监测点位图缺失或点位与坐标不符 → P1
- 特征因子未监测且无合理解释 → P1
- 超标因子未分析原因 → P2
- 引用 3 年前数据 → P1
