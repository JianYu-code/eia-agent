# K15 附图附件完整性审核要点

## 报告书必备附图
1. 地理位置图（含行政区划、水系）
2. 厂区平面布置图（标注环保设施位置）
3. 周边环境关系图/敏感目标分布图（标注评价范围、防护距离）
4. 环境现状监测布点图
5. 区域水系图（涉及地表水评价时）
6. 土地利用/规划图（规划符合性佐证）
7. 生态红线位置关系图（涉及时）
8. 地下水/土壤评价范围图（设专项时）

## 必备附件
1. 项目立项/备案文件
2. 委托书
3. 现状监测报告（CMA）
4. 营业执照/法人证明
5. 土地证明/租赁协议
6. 危废处置协议（或意向书）
7. 公众参与材料（报告书）
8. 现有工程环评批复+验收意见（改扩建项目）
9. 总量指标来源文件

## 审核要点
- 附图仅列目录未见实际图件描述 → P1（提示核对纸质版）
- 附图缺必备项（如平面布置图）→ P1
- 改扩建项目缺现有工程批复/验收附件 → P1
- 附件与正文引用不一致（附件编号、日期）→ P2
