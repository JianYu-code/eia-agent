# K02 分类管理名录判定要点

## 依据
《建设项目环境影响评价分类管理名录（2021年版）》

## 判定逻辑
1. **行业类别定位**：按《国民经济行业分类》确定项目行业门类+小类，再查名录对应条目。
2. **等级判定**：名录条目区分 报告书/报告表/登记表 三档：
   - 可能造成重大环境影响 → 报告书
   - 轻度影响 → 报告表
   - 影响很小 → 登记表（备案制，无需审批）
3. **敏感区升级**：涉及环境敏感区（自然保护区、风景名胜区、饮用水水源保护区、生态保护红线等）的项目，名录中标注"涉及环境敏感区"条款的，应按更高等级编制。
4. **复合项目**：同一项目跨多个类别时，按其中最高等级执行（就高不就低）。
5. **扩建/技改**：按名录中对应行业的新建/扩建条目判定；技改后污染物种类或量增加的不得降级。
6. **过度评价也是问题**：名录允许报告表而编制了报告书 → P2（增加企业负担）。

## 常见误判点
- 规模阈值（如"年出栏生猪5000头"）对应的实际产能核算错误
- 新建/扩建界定错误导致条目选择错误
- 忽视"涉及环境敏感区"触发升级
- 审批权限：报告书由地市级及以上审批（部分行业省级），与等级判定联动核查

## 输出要求
- 等级判定错误 → P0
- 敏感区未升级 → P0
- 等级正确但未论证判定过程 → P1
