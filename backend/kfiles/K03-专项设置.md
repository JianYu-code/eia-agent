# K03 专项设置审核要点

## 专项评价设置条件（依据各要素导则）

### 大气专项（HJ 2.2-2018）
- 排放 SO₂+NOx ≥ 500 t/a，或排放污染物含《有毒有害大气污染物名录》物质，或评价等级一级的项目应设大气专项。

### 地表水专项（HJ 2.3-2018）
- 直接向水体排放污水的项目（直接排放）应设地表水评价专题；间接排放可简化。

### 地下水专项（HJ 610-2016）
- Ⅰ类、Ⅱ类建设项目在地下水环境敏感区，或污水产生量大的项目应设地下水专项；报告表项目原则上不开展，特殊情况（垃圾填埋场、危废处置场）除外。

### 土壤专项（HJ 964-2018）
- 污染影响型项目涉及重金属、持久性有机污染物或位于耕地等敏感区应开展土壤评价。

### 生态专项（HJ 19-2022）
- 涉及生态敏感区（国家公园、自然保护区、生态保护红线）或工程占地规模大的项目应设生态专题。

### 环境风险专项（HJ 169-2018）
- 涉及危险物质且 Q≥1 的项目应开展风险评价；Q<1 可简单分析。风险潜势Ⅲ及以上应设专项。

### 辐射专项
- 涉及电磁辐射（变电站、输电线）或伴生放射性矿的项目应设辐射专题（HJ 1114、HJ 10.2 等）。

## 审核要点
1. 应设专项而未设 → P0
2. 专项评价等级判定错误（详见 K09）→ P0
3. 专项内容深度不足（缺预测、缺监测计划）→ P1
4. 不应设而设置（报告表项目硬套专项）→ P2
