# K12 危险废物管理审核要点

## 依据
《国家危险废物名录（2021年版）》、GB 18597-2023《危险废物贮存污染控制标准》

## 常见危废速查
| 废物 | 类别代码 |
|---|---|
| 废活性炭（吸附有机废气/VOCs） | HW49 900-039-49 |
| 废机油、废润滑油 | HW08 900-214-08 / 900-217-08 |
| 废漆渣、废油漆桶 | HW12 900-252-12 |

## 审核要点
1. **识别完整性**：对照工艺逐项排查（设备维修→废机油；废气处理→废活性炭/废催化剂；喷漆→漆渣）遗漏 → P0
2. **属性判定**：名录外废物需做鉴别（GB 5085 系列），不能简单按一般固废处理
3. **估算量**：危废产生量估算及依据 → 缺失 P1
4. **贮存**：危废暂存间防风防雨防渗（渗透系数 ≤10⁻¹⁰ cm/s）、分区分类、标识标牌、台账 → 缺失 P1
5. **处置去向**：委托有资质单位处置，附处置协议或意向 → 缺失 P0
6. **转移**：转移联单制度、跨省转移审批 → 缺失 P2
7. 一般固废与危废混存、危废混入生活垃圾 → P0

## 注意
- 废活性炭代码易错：吸附 VOCs 的为 HW49 900-039-49，不是 HW06
- 污泥属性取决于来源：电镀污泥 HW17，生活污泥一般为一般固废，含油废水污泥 HW08
