# K04 工程分析与源强核算审核要点

## 依据
《污染源源强核算技术指南 准则》（HJ 884-2018）及各行业源强核算技术指南（HJ 888~HJ 996 系列）

## 核算方法优先次序
1. **物料衡算法**（新建项目优先）：基于原辅料成分、反应方程式核算
2. **类比法**（改扩建项目常用）：须满足全部类比条件
3. **产污系数法**（有系数手册的行业）：系数须有出处
4. **实测法**（现有工程）

## 类比法 5 个必要条件（缺一即不成立）
1. 原料、燃料类别相同，且与污染物产生相关的成分相似（差异 ≤10%）
2. 辅料类型相同
3. 生产工艺相同
4. 产品类型相同
5. 生产规模差异 ≤30%

## 审核要点
1. **污染因子完整性**：对照行业指南与排放标准，是否遗漏特征因子：
   - 锅炉：汞及其化合物、烟气黑度、氨逃逸（SCR 脱硝）
   - 化工：VOCs、特征有机物、重金属
   - 废水：特征污染物（第一类污染物须车间排放口达标）
   - 固废：危废识别遗漏（详见 K12）
2. **无组织排放**：储罐、装卸、污水站、危废间等无组织源是否识别
3. **非正常工况**：开停车、检修、事故状态下的源强是否给出
4. **取值出处**：每个源强数据须注明来源（类比项目名/实测报告/系数手册页码）
5. **三本账**：改扩建项目须给出现有工程、拟建工程、以新带老削减的"三本账"
6. **物料平衡**：主要元素（S、N、重金属）应有平衡分析
7. 核算时段：小时最大排放量（用于预测）与年排放量（用于总量）是否区分

## 典型 P0 情形
- 源强无任何出处依据
- 类比法不满足条件仍使用
- 遗漏排放标准中受控的特征因子
- 改扩建无三本账
