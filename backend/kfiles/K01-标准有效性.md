# K01 标准有效性审核要点

## 适用范围
核查环评/验收/应急报告中引用的标准、导则、规范是否为现行有效版本。

## 审核要点
1. **版本完整性**：标准引用必须含标准号+年号（如 HJ 2.1-2016）。只写标准号不写年号，视为引用不规范（P2）。
2. **废止判定**：引用已废止标准 → P0。常见已废止：
   - GB 16297-1996《大气污染物综合排放标准》部分行业已被行业型标准替代（GB 37822-2019、GB 39726-2020、GB 41616-2022 等）
   - GB 8978-1996《污水综合排放标准》仍现行但行业排放优先执行行业标准
   - HJ/T 2.1~2.4-93 系列已被 HJ 2.1-2016、HJ 2.2-2018、HJ 2.3-2018、HJ 2.4-2021 替代
   - HJ/T 169-2004 已被 HJ 169-2018《建设项目环境风险评价技术导则》替代
   - GB 3095-1996 已被 GB 3095-2012《环境空气质量标准》替代
   - GB 3838-2002《地表水环境质量标准》现行有效
   - GB 12348-90 已被 GB 12348-2008《工业企业厂界环境噪声排放标准》替代
3. **替代关系**：注意"部分替代"——综合标准与行业标准并存时，有行业标准的优先执行行业标准。
4. **地方标准**：DB 开头的地标（如 DB11/、DB32/、DB44/）在其行政区域内优先于国标；核查适用区域是否匹配。
5. **新旧过渡**：新标准发布后有过渡期的，注意项目实施日期对应的适用版本。
6. **导则类**：HJ 2.1-2016 总纲、HJ 2.2-2018 大气、HJ 2.3-2018 地表水、HJ 610-2016 地下水、HJ 964-2018 土壤、HJ 19-2022 生态、HJ 169-2018 风险、HJ 2.4-2021 声环境，注意使用最新版。

## 输出要求
- 发现废止标准 → P0，必须给出替代标准号
- 缺年号/版本不明 → P2
- 不确定时效性时写"请核实 XX 现行有效性"，不得臆造替代关系
