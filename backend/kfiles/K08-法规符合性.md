# K08 法规与规划符合性审核要点

## 审核维度
1. **产业政策**：《产业结构调整指导目录》——禁止类不得建设，限制类不得新建；淘汰类工艺/设备 → P0
2. **三线一单**（生态保护红线、环境质量底线、资源利用上线、生态环境准入清单）：
   - 项目选址是否在生态保护红线内（一般生态空间/红线区）→ 红线内禁止 → P0
   - 是否符合所在环境管控单元的准入清单
3. **规划符合性**：
   - 是否符合城市总体规划、土地利用规划、园区规划
   - 园区内项目是否符合园区规划环评及审查意见的产业定位与准入条件
4. **选址合理性**：防护距离（大气环境防护距离、卫生防护距离）内是否有敏感目标
5. **两高项目**：高耗能高排放项目是否符合区域削减、能耗双控要求
6. **流域/区域政策**：长江保护法、黄河保护法、太湖流域管理条例等对化工、印染项目的限制
7. **VOCs/重金属 特别政策**：重点行业综合治理方案、重金属总量控制

## 审核要点
- 未开展三线一单符合性分析 → P1（报告书）/ P2（报告表）
- 引用已失效的产业政策版本 → P1
- 位于禁建区（水源一级保护区、自然保护区核心区）→ P0
- 园区项目未对照规划环评准入条件 → P1
